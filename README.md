# イントロダクション
SFC開講「統計解析」のための演習用資料ページです。

`git2r`パッケージ等を利用して, このレポジトリをクローンして利用することをお勧めします。
ここでは, `exercise`というフォルダ名でレポジトリをクローンします。

```r
install.packages("git2r")
url <- "https://github.com/N-Yukihiro/statistical_analysis_SFC.git"
git2r::clone(url = url,
             local_path = "exercise")
```
